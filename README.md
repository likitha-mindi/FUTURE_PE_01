# FUTURE_PE_01  
## AI Website Copy Generator for Local Businesses  

### Internship Track:
Prompt Engineering – Future Interns  

## Project Overview  
This project demonstrates the use of structured prompt engineering to generate high-quality website copy for local businesses using AI tools.  
The objective is to create:
- Homepage copy
- Service page descriptions
- Strong Call-to-Action (CTA) sections  
All prompts, outputs, and explanations are documented in this repository.

## Tools Used  
- ChatGPT  
- Google Docs  
- GitHub  

## Business Chosen  
Local Cafe – Urban Brew Cafe 
Target Audience: Young professionals, students, and families  
Location Focus: Local neighborhood customers  
Tone: Warm, modern, inviting

## Repository Structure  
- prompts.md → Contains structured prompts used  
- outputs.md → Contains AI-generated outputs  
- explanation.md → Contains reasoning behind prompt structure  

## Learning Outcome  
- Prompt structuring for business use  
- Conversion-focused website copywriting  
- AI workflow documentation
