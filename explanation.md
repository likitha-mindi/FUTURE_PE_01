# PROMPT ENGINEERING EXPLANATION

## Objective
To generate structured, conversion-focused website copy for a local café using AI.

## Prompt Strategy
1. Role Assignment
The AI was instructed to act as a "professional website copywriter" to ensure business-quality output.

2. Clear Structure
The prompt specified exact sections:
- Hero headline
- Value proposition
- Services breakdown
- CTA sections
This reduces vague output and ensures organized content.

3. Target Audience Definition
Specifying students, professionals, and families helps AI tailor tone and messaging.

4. Tone Specification
Words like warm, inviting, modern guide the writing style.

5. Conversion Focus
Separate CTA prompt ensures persuasive and action-oriented messaging.

## Outcome
The AI generated structured, website-ready content that:
- Is simple and readable
- Focuses on customer benefits
- Encourages action
- Matches a real local café business use-case
## Prompt Techniques Used
1. Keyword Targeting  
Primary and secondary keywords were added to improve SEO optimization.

2. Word Count Control  
Specifying 300 words ensures structured, substantial output.

3. Competitor Positioning  
The AI was instructed to differentiate from competitors to simulate real business copywriting strategy.

4. Conversion Psychology  
CTA emphasis ensures business-oriented output rather than generic content.
